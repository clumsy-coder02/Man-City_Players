# Manchester City Players

A Python command-line application that displays Manchester City football players organized by their positions on the field as of 2026.

## Description

This application allows users to browse through Manchester City's squad by selecting different positions. The players are categorized into:

- **Goalkeepers (GK)**
- **Defenders** (Right Back, Left Back, Center Back)
- **Midfielders** (Defensive, Central, Attacking)
- **Attackers** (Left Wing Forward, Center Forward, Right Wing Forward)

## Features

- Interactive menu system for position selection
- Detailed player listings for each position
- Command-line interface
- Comprehensive squad coverage

## Requirements

- Python 3.x

## How to Run

1. Clone or download the repository
2. Navigate to the project directory
3. Run the application:

```bash
python app.py
```

4.Follow the on-screen prompts to select positions and view players

## Project Structure

- `app.py`: Main application script
- `info.txt`: Project documentation and notes
- `README.md`: This file

## Usage

When you run the app, you'll see:

```bash
Manchester City Players
1.GoalKeeping
2.Defense
3.Midfield
4.Attack

Select player's position from above?
```

Choose a number (1-4) to explore players in that category. For defenders, midfielders, and attackers, you'll be prompted to select sub-positions.
